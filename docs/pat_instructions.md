# Instructions pour générer un Personal Access Token (PAT)

## Procédure
1. Connectez-vous à GitHub → Settings → Developer settings → Personal access tokens.
2. Cliquez sur Generate new token (classic).
3. Donnez un nom descriptif au token (ex. "Test Token Exercice").
4. Choisissez une expiration courte (ex. 30 jours).
5. Sélectionnez uniquement les scopes nécessaires.
6. Cliquez sur Generate token.
7. Copiez le token dans un endroit sûr (ne jamais le commiter).

## Scopes choisis
- `repo` : accès au dépôt (lecture/écriture)
- `workflow` : pour exécuter des actions GitHub
- `read:user` : lecture des informations utilisateur
