Nous avons choisi de rendre le dépôt **public** pour permettre au professeur et aux collaborateurs de vérifier le projet facilement, sans restriction d'accès. Le projet contient des fichiers de configuration et peut inclure des données sensibles liées aux analyses et à l’authentification.

# Décision de visibilité du dépôt

Le dépôt est public car :
- Il ne contient pas de données sensibles.
- Le code est destiné à être partagé et évalué par l’enseignant et les étudiants.
- Cela permet également de rendre le projet consultable pour l’apprentissage ou contributions futures.
