# Quickstart GitHub : Créer une branche et une Pull Request

## 1️⃣ Aller sur ton dépôt
Connecte-toi sur GitHub et ouvre ton dépôt.

## 2️⃣ Créer une nouvelle branche
- Clique sur le menu déroulant **branch: main**
- Tape le nom de ta branche, par exemple `feature/ma-fonctionnalite`
- Clique sur **Create branch: feature/ma-fonctionnalite**

## 3️⃣ Modifier ou ajouter des fichiers
- Clique sur **Add file → Create new file** ou modifie un fichier existant
- Ajoute ton contenu
- Dans **Commit changes**, mets un message clair et clique sur **Commit directly to feature/ma-fonctionnalite**

## 4️⃣ Créer la Pull Request
- Clique sur **Compare & pull request**
- Renseigne le **titre** et la **description**
- Clique sur **Create pull request**

## 5️⃣ Attendre l’approbation
- Au moins un reviewer doit approuver
- Ensuite le bouton **Merge pull request** sera disponible

## 6️⃣ Fusionner et supprimer la branche
- Clique sur **Merge pull request → Confirm merge**
- Clique sur **Delete branch** si tu veux supprimer ta branche feat
